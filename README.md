# 📱 QR File Reconstructor - 二维码文件重组工具
一个通过扫描多张二维码，将分片数据重组为原始文件的 Android 应用。
## ✨ 功能特性
- ✅ 扫描二维码获取 JSON 分片数据
- ✅ 支持乱序扫描，自动排序
- ✅ Base64 解码，支持任意文件格式
- ✅ MD5 校验和验证，确保数据完整性
- ✅ 自动保存到手机存储
- ✅ 简洁易用的用户界面
## 📦 数据格式
```json
{
  "totalPackages": 7,
  "currentPackageNumber": 2,
  "encoding": "Base64",
  "dataLength": 2000,
  "dataContent": "...",
  "fileName": "README.md",
  "checksum": "18bee1de046b63aad34538e97aa631c4"
}
```
## 🚀 快速开始
### 方法一：GitHub Actions 自动编译
1. 访问仓库的 Actions 页面
2. 等待编译完成（约 2-5 分钟）
3. 下载生成的 APK 文件
4. 安装到 Android 手机
### 方法二：本地编译
1. 克隆仓库：`git clone https://github.com/luomucc/QRFileReconstructor.git`
2. 用 Android Studio 打开 `QRFileReconstructor` 文件夹
3. 连接手机或启动模拟器
4. 点击运行按钮
## 📱 使用方法
1. 打开应用，授予相机权限
2. 点击"开始扫描下一张"按钮
3. 依次扫描所有二维码（无需按顺序）
4. 扫描完成后，应用自动重组文件
5. 文件保存到 `Download/QRReconstructor/` 目录
## 🛠️ 技术栈
- **语言**: Java
- **扫码库**: ZXing
- **构建工具**: Gradle
- **目标 SDK**: Android 7.0+ (API 24+)
